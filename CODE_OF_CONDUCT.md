# Código de Conducta

Este proyecto sigue el [Contributor Covenant v2.1](https://www.contributor-covenant.org/es/version/2/1/code_of_conduct/). Se espera que todas las personas que participan en este proyecto respeten las siguientes normas básicas:

- Sé respetuoso y considerado con los demás.
- No toleramos lenguaje o comportamiento discriminatorio u ofensivo.
- Ten en cuenta que el comportamiento de fuera de esta comunidad también puede afectar a las relaciones dentro de ella.

## Cómo reportar
Si experimentas o presencias un comportamiento que viola este código, por favor contacta a los mantenedores del proyecto a través de `conduct@kevindels.dev` (reemplaza por un contacto real) o abre una issue marcada como `code-of-conduct`. Todos los reportes serán revisados con confidencialidad.

## Aplicación
Los mantenedores del proyecto se reservan la opción de tomar medidas apropiadas en caso de conductas que infrinjan estas normas, incluyendo advertencias y prohibición de contribuciones.

Gracias por ayudar a crear una comunidad inclusiva y respetuosa.
