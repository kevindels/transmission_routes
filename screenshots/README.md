# Screenshots

Guarda aquí capturas de pantalla y GIFs para mostrar en el `README.md`.

Sugerencias:

- Formato: `.png`, `.jpg` o `.gif` (para demos animadas)
- Ancho recomendado: 800–1200 px para screenshots; para GIFs, trata de mantenerlos ligeros (≤ 3–5 MB)
- Nombres: `login.png`, `home.png`, `streaming.png`, `history.png`, `demo.gif`
- Añade una entrada en el `README.md` que apunte a estas imágenes, por ejemplo:

```markdown
![Streaming demo](screenshots/demo.gif)
```

Puedes usar `ffmpeg` o herramientas como LICEcap para crear GIFs de demostración.
